Enc REST API for doing String Encryption and Decryption

Available methods -

1. /api/encrypt (POST)

Input: 

{"input": "somevalue"}

Output:

{
"Input": "My string to encrypt",
"Output": "DKJALDKLKADF===",
"Status": "success",
"Message": ""
}

2. /api/decrypt (POST)

Input: 

{"input": "somevalue"}

Output:

{
"Input": "DKJALDKLKADF===",
"Output": "My string to encrypt",
"Status": "success",
"Message": ""
}

3. /api/health (GET)

Output: 

{"state": "healthy"}
